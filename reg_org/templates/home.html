Hello, This is an Identity and Access Management System (IAM) , which can be used by different Organizations to manage their Users.

How Can You Do it ?

Steps:-
  • Regiser as an Organizations
  • Add permission Levels
  • Add Users of your Organization
  • Add Permissions to different Users
How is it Done?
It uses Token (specifically JWT tokens) for authentication and authorization. It is like any other Oauth , which generates a token with Restricted access which can be configured by the organization .Hence ensuring not all the data is exposed to third party
